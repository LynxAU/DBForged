## PART 2: New Feature Ideas (From Research + Creative)

### TIER 1 - Core Gameplay Enhancements

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **Combo System** | Chain attacks: punch → punch → kick → special | Fighting game feel |
| **Ki Control Training** | Mini-game: hold ki at specific levels | DBZ training vibes |
| **Flying Training** | Navigate through rings for XP | Classic DBZ |
| **Mentor System** | Learn ultimate techniques from master NPCs | Progression depth |

### TIER 2 - Social & Economy

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **Player Shops** | Set up vendor stalls in cities | Player economy |
| **Trading System** | Secure trade between players | Social |
| **Friends List** | Add players, see online status | Community |
| **Global Chat** | Trade, LFG, OOC channels | MUD social |
| **Mail System** | Send items/zeni to offline players | Convenience |

### TIER 3 - Endgame

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **Ultimate Tower** | Infinite scaling floor climb | Endgame goals |
| **Weekly Raids** | Rotating boss events | Live game feel |
| **Season Reset** | PvP seasons with rewards | Competition |
| **Leaderboards** | Top PL, kills, wins | Motivation |

### TIER 4 - Character Expression

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **More Costumes** | Alternate outfits, gi colors | Customization |
| **Z Encyclopedia** | Collectible lore entries | Collecting |
| **Title System** | Earn titles from achievements | Status |
| **Pose/Emotes** | Battle poses, victory animations | Expression |
| **Custom Aura** | Choose aura color/style | Identity |

### TIER 5 - Advanced Combat

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **Guard/Block** | Defensive mechanics | Strategy |
| **Counter Attack** | Punish missed attacks | Skill |
| **Ki Cancel** | Spend ki to cancel animations | Tech |
| **Rage Mode** | Low HP = boosted damage | Comebacks |
| **Limit Break** | Push beyond max PL temporarily | Epic moments |

### TIER 6 - World Building

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **Housing** | Personal room/space | Ownership |
| **Capsule House** | Deployable base | DB style |
| **Clan Base** | Shared faction headquarters | Group goals |
| **Territory Wars** | Control regions of map | Faction PvP |

### TIER 7 - Events & Seasons

| Feature | Description | Why Cool |
|---------|-------------|----------|
| **Holiday Events** | Christmas, Halloween specials | Fresh content |
| **Tournament Events** | Weekly PvP tournaments | Competition |
| **World Events** | Random world announcements | Excitement |
| **Anniversary** | DBForged birthday event | Community |
